	-npm 함께설치
	-npm : 다양한 외부 패키지들의 버전과 의존성을 관리하고 편하게설치및 삭제 역할
		-React 설치하려고
1.터미널 실행
2.현재프로젝트 경로확인
3.cd src/main 이동
4. npx create-react-app frontend < 프론트앤드는 내가 지은거임 아무 프로젝트명 해도됩니다 
5.cd frontend
6.npm start
7.3000번대 뜨면 일단 설치 완료
